Java-使用数据流对文件的复制

  • Java
  • 笔记

数据流的输入与输出都是对应的，在此仅以输入类做为说明。

一般常用InputStream类型如下：
ByteArrayInputStream：允许将内存的缓冲区当输入流，一般与FilterInputStream对象相连；
StringBufferInputStream：可将字符串转换成输入流；
FileInputStream：可以将文件转换成输入流；

一般常用FilterInputStream类型如下：
DataInputStream：以基本数据类型进行读；
BufferedInputStream：使用缓冲区进行读；

上方是以字节方式进行，下方以字符方式进行。

一般常用的Reader类型：
FileReader，StringReader，CharArrayReader；

一般常用的FilterReader类型：
BufferedReader，PrintWrite；
